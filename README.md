## Core Objective
A high-performance, aesthetically futuristic web application that uses computer vision to classify waste and provide professional disposal, recycling, and upcycling intelligence. 

**Deployed System**: [EcoSort on Streamlit Cloud](https://streamlit.io) *(Your Link Here)*

*   **3Rs Strategy Engine**: Provides Recyclability scoring, disposal guides, and upcycling projects.
